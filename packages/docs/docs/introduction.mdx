# Introduction

Claude Codex is a multi-agent development environment that helps you manage git worktrees, tmux sessions, and development workflows.

## Features

- **Git Worktree Management**: Create and manage isolated development environments
- **Tmux Integration**: Seamlessly handle terminal sessions
- **GitHub Integration**: Connect with GitHub issues and repositories  
- **Web UI**: Modern terminal interface for visual management
- **Library-First Design**: Extensible core modules for custom integrations

## Architecture

Claude Codex follows a 4-layer architecture:

```
commands/ → workflows/ → core/ → shared/
```

- **commands/**: CLI entry points
- **workflows/**: Orchestrate multiple operations  
- **core/**: Reusable building blocks
- **shared/**: Common utilities and types
