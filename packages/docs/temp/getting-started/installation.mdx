# Installation

*Complete installation guide for all platforms*

## System Requirements

### Prerequisites
- Node.js 18+ or Bun runtime
- Git 2.30+
- tmux 3.0+
- GitHub account and token

### Platform Support
- macOS (recommended)
- Linux (Ubuntu, Debian, Arch)
- Windows (WSL2)

## Installation Methods

### NPM Installation
```bash
npm install -g claude-codex
```

### Bun Installation
```bash
bun install -g claude-codex
```
