# API Reference

*Complete API documentation for Claude Codex*

## Overview

Claude Codex provides multiple API layers for different use cases:
- **Core API**: Library functions for direct integration
- **MCP API**: Model Context Protocol for agent coordination
- **REST API**: HTTP endpoints for web interface
- **WebSocket API**: Real-time terminal streaming

## API Categories

### [Core API](core-api.md)
**Library functions and TypeScript interfaces**
- Worktree management functions
- GitHub integration methods
- tmux session operations
- Claude Code integration
- File and configuration management

### [MCP API](mcp-api.md)
**Model Context Protocol tools and resources**
- Agent coordination tools
- Instance management resources
- Status update mechanisms

### [REST API](rest-api.md)
**HTTP endpoints for web interface**
- Instance CRUD operations
- Configuration management
- Status and monitoring endpoints

### [WebSocket API](websocket-api.md)
**Real-time communication protocols**
- Terminal streaming
- Agent status updates
- Live progress monitoring
